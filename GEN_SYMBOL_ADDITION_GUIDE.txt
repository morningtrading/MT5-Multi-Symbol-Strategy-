================================================================================
                           TROUBLESHOOTING GUIDE
================================================================================

üö® PROBLEM: Symbol shows "‚ùå Not available" during screening
üí° SOLUTION:
   - Verify symbol name spelling in MT5 Market Watch
   - Check if symbol is available during current market hours
   - Ensure MT5 is connected and logged in
   - Try refreshing MT5 Market Watch

üö® PROBLEM: Risk manager rejects symbol "not in risk configuration"
üí° SOLUTION:
   - Double-check symbol name matches exactly (case-sensitive)
   - Verify JSON syntax is correct (no trailing commas)
   - Ensure proper placement in position_coefficients section

üö® PROBLEM: Order execution fails
üí° SOLUTION:
   - Check min_lot value matches symbol specifications
   - Verify sufficient account balance
   - Ensure market is open for the symbol
   - Check MT5 connection stability

üö® PROBLEM: JSON format errors
üí° SOLUTION:
   - Use online JSON validator to check syntax
   - Common issues: missing commas, trailing commas, unmatched brackets
   - Ensure all strings are in double quotes "like this"

   - Increase gradually based on performance
   - Higher coefficients = larger position sizes

üéØ ASSET CLASS OPTIMIZATION

   - Major forex: EURUSD, GBPUSD, USDJPY, AUDUSD
   - Crypto: All cryptocurrency pairs

   - Consider correlation between similar instruments

   - Adjust coefficients based on volatility
